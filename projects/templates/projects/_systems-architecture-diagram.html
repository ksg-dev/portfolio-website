Each template loads CSS in this order:

  1. Base CSS - From base.html (global AURA styles)
  2. systems.css - Base systems styling
  3. systems-unified.css - Glass container components
  4. Template-specific CSS - dashboard.css, system_list.css, etc.
  5. Inline styles - Template-specific overrides

Responsive Behavior

Desktop (>1200px)

  • Dashboard: 2fr 1fr grid
  • List: auto-fill minmax(350px, 1fr)
  • Detail: 2fr 1fr grid
  • All panels maintain full features

Tablet (768px-1200px)

  • Dashboard: maintains 2-column
  • List: fewer columns in grid
  • Detail: maintains 2-column
  • Metrics grid adapts to smaller cards

Mobile (<768px)

  • Dashboard: stacks to 1 column
  • List: single column layout
  • Detail: stacks to 1 column
  • Navigation becomes collapsible
